**ShipsecAI Studio** is a modern, open source automation platform built for security and IT engineers. Shipsec  comes with a no-code / low-code UI for workflows, out-of-the-box integrations, lookup tables, and case management. Workflows are orchestrated using [**Temporal**](https://temporal.io/) for scale and reliability.

## Get Started

<Tip>
  We highly recommend every user complete the [**quickstart tutorial**](https://docs.tracecat.com/quickstart/tutorial). This tutorial covers all the core features of ShipsecAI Studio, which will save you hours of learning time.
</Tip>
