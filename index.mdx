**ShipSec Studio** is an open source automation platform built for security and IT engineers. It provides a no-code/low-code UI for building workflows, out-of-the-box integrations, and reliable orchestration powered by [Temporal](https://temporal.io/).

## Why ShipSec Studio?

- **Component-driven orchestration** – Drag-and-drop discovery, DNS, HTTP, and enrichment components
- **Temporal reliability** – Every workflow runs with retries, heartbeats, and resumability baked in
- **Live execution visibility** – Real-time traces, logs, and progress streaming
- **Single-machine friendly** – Full stack runnable on a laptop with Docker Compose
